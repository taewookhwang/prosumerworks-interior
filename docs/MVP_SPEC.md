# 인테리어 시공 중개 플랫폼 MVP 명세서

## 1. 서비스 개요

### 비전
아파트 인테리어 시공업체와 고객을 연결하는 중개 플랫폼

### 핵심 가치
- **고객**: 실제 시공 사례를 보고 믿을 수 있는 업체 찾기
- **업체**: 포트폴리오로 고객 유치
- **플랫폼**: 계약 성사 시 중개 수수료

### 타겟 시장
- 지역: 한국
- 고객: 아파트 인테리어 (전체/부분 시공)
- 시공 범위: 전체 리모델링, 욕실, 주방, 발코니, 페인트 등

---

## 2. 사용자 유형

### 2.1 고객 (Customer)
- 인테리어 시공을 원하는 아파트 거주자
- 주요 니즈: 시공 사례 탐색, 업체 비교, 견적 요청

### 2.2 시공업체 (Contractor)
- 인테리어 시공 전문 업체
- 주요 니즈: 포트폴리오 홍보, 고객 유치, 견적 응답
- 가입 조건: 사업자등록증 + 포트폴리오 제출 → 관리자 승인

### 2.3 관리자 (Admin)
- 플랫폼 운영자
- 역할: 업체 심사/승인, 계약 관리, 수수료 정산

## 5. 수익 모델

### 중개 수수료
- **정산 시점**: 계약 성사 시
- **수수료율**: 계약금의 10% (조정 가능)
- **정산 방식**: MVP에서는 수동 정산, 추후 자동화

### 향후 확장
- 프리미엄 업체 광고
- 상단 노출 광고
- 프리미엄 구독 (업체)

---

## 6. 화면 구성 (Screens)

### 공통
1. 스플래시
2. 로그인 (Google)
3. 회원유형 선택
4. 업체 가입 신청 폼

### 고객 (탭 구조)
1. **홈**: 시공 사례 피드
2. **검색**: 카테고리 + 검색
3. **저장**: 저장한 사례
4. **채팅**: 채팅 목록
5. **마이**: 프로필 + 견적 요청 내역

### 업체 (탭 구조)
1. **홈**: 받은 견적 요청
2. **포트폴리오**: 내 시공 사례
3. **업로드**: 시공 사례 등록
4. **채팅**: 채팅 목록
5. **마이**: 업체 프로필 + 계약 내역

## 9. 성공 지표 (KPI)

### Launch 목표
- 등록 업체: 50개
- 시공 사례: 200개
- 가입 고객: 500명
- 첫 계약 성사: 10건

### 핵심 지표
- 견적 요청 전환율 (피드 → 요청)
- 견적 응답률 (요청 → 응답)
- 계약 성사율 (응답 → 계약)
